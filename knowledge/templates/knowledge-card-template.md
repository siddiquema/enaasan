# Knowledge Card Template

For a single, atomic, reusable unit of knowledge — a term, concept, or short explanation that can be embedded or referenced across many other documents (a Guide, a Glossary, an FAQ answer, a Career Profile) without being rewritten each time. See `standards/knowledge-card-standard.md` for the full standard this template implements — read that file before creating a real card.

Copy the block from `metadata-template.md` above this line, with `document_id` using type `KCARD`. Knowledge Cards use `estimated_reading_time` (should almost always be well under a minute).

---
