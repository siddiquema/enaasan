# Guide Template

For a plain-read explanatory document — the most general-purpose publication type. Also used for **Reference Guides** (a Guide that's meant to be consulted/searched rather than read start-to-finish — mark this by setting `reading_level` to note its reference nature and keeping sections more self-contained, so a reader can jump to one section without needing the others).

Copy the block from `metadata-template.md` above this line, with `document_id` using type `GUIDE` or `REFGUIDE`.
