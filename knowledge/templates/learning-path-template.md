# Learning Path Template

For a sequenced set of documents/Knowledge Cards toward a defined goal (e.g. "understand your options after 12th Commerce" or "get ready for entry-level data analytics roles"). A Learning Path doesn't contain much original content itself — it's primarily a curated, ordered set of pointers to other documents, with connective narrative explaining why this order and what each step is for.

Copy the block from `metadata-template.md` above this line, with `document_id` using type `LPATH`. Use `estimated_completion_time`.

---

[One or two sentences: what goal this path leads toward, and who it's for. Be specific about the starting point assumed — see `prerequisites` in the metadata block.]

## Where this path leads

[A short paragraph describing the destination state — what the reader will understand or be able to do once they've worked through the path. Avoid vague outcomes ("you'll know more about X"); be concrete.]

## After this path
