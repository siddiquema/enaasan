# publications/graduates/

## Purpose

Real publications aimed at recent graduates and final-year students — job search, reskilling, higher-studies decisions, workplace entry. Corresponds to the `GRAD` audience code in `architecture/document-id-standard.md` and to the `/graduates` topic hub concept in `docs/website-update-spec.md` §10.

## Contents

| Publication | Document ID |
|---|---|
| `first-90-days-job-search/` | ENA-WORKBOOK-GRAD-first-90-days-job-search-v1.0 — the definitive first-job-search workbook (project ENA-GRAD-001) |

Each publication gets its own subfolder, named by slug.

## Naming Convention

`publications/graduates/<slug>/source.md`, where `<slug>` matches the Document ID's slug exactly.

## Rules

Same as the parent `publications/README.md` — Document ID, complete metadata, and a followed template are required before anything lands here.

## Examples

- The existing `assets/enaasan-90-day-career-clarity-plan.pdf` is the natural first candidate for formal onboarding here, as `publications/graduates/90-day-career-clarity-plan/source.md` (see `publications/README.md`'s example).

## Future Expansion

- Likely early content: Career Profiles for common graduate-entry roles, a Learning Path for "first 90 days in a new job," Knowledge Cards for terms graduates commonly encounter (CTC, notice period, probation).
